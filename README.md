# Advanced-Database System
Implemented various full stack websites as a project
Project 1 implements various database functionality which can be queried according to the user's specification.
Project 2 implements various queries on database and optimise it using the redis caching.
Project 3 provides the results of the queries in the visual format.
Project 4 implements some Natural Language Processing tasks on the file provided by the user.
Project 5 implements an interface for a student and teacher for an exam which can be moderated by the admin.
